Any range ("rng") value can be:

    * a single value (e.g. 1)
    * a range (e.g. [1,10] or (1,10))
    * a dictionary with keys and probabilities (e.g. {1:0.5, 2:0.5})
    * Examples
            {"mean": 1, "sd": .1, "max": 2, "min": 0}
            {"mean": 1, "sd": .1}
            {1: 1, 2: 1, 3: 1}
